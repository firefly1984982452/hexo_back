# 夜神模拟器连接HBuilder

1.进入夜神模拟器bin目录

2.输入命令

```
nox_adb connect 127.0.0.1:62001

nox_adb devices
```

3.进入HBuilder的tools目录

4.以下命令
```
adb connect 127.0.0.1:62001
adb devices

```

5.还是不行的话在HBuilder中设置

工具 => 选项 => HBuilder => 第三方Android模拟器端口

改成：`62001`
